CSSPropertiesConstants
 
Packagecom.flashartofwar.fcss.enum
Classpublic class CSSProperties
InheritanceCSSProperties Object

This class contains a lists of "supported" css properties that FSS could build off of. This is meant to be used as a reference when coding and is not used at runtime.
